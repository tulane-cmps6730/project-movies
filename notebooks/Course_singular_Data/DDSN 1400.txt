DDSN 1400Typography Studio I(3)This studio course examines typography from historical and practical perspectives while creating connections between type as a communication tool and type as visual expression. Topics introduced in this course are the history of type, type anatomy, classifications, terminology, type setting for the screen versus print, rules of typography, hierarchy and formalism. Additionally, students engage in written and verbal communication for ideation and presentation. Lectures, readings, class demonstrations, group exercises and writing assignments are used. 
Prerequisites:DDSN 1100,1101,1102.Prerequisite(s):DDSN 1102.
