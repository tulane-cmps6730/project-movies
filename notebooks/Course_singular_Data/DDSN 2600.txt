DDSN 2600Interactive Design Studio I(3)This studio course explores design for mobile, display, aesthetics, UX, UI and interactive solutions. Students are introduced to the latest authoring and animation tools and learn how to apply their knowledge of the software to create dynamic and responsive interactive experiences. Students will communicate ideas via written and verbal presentation.
Prerequisites:DDSN 1100,1101,1102,1401,2100,2101,CPST 3400,CPST 3410.Prerequisite(s):DDSN 1401,1400and2101.
