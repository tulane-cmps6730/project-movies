CMPS 2120Programming and Prob Solving(3)An introductory course on computer programming and problem-solving using computers. In this course students design, implement, test and debug programs for computational problems using Python programming language. This course emphasizes program design process, object-oriented software development approach, and practical programming skills that translate to programming in other modern languages. Assignments include practical problems drawn from various fields (such as biology, linguistics, graphics, and games).
