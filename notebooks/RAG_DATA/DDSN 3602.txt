DDSN 3602Interactive Design Studio II(3)This studio courses uses advanced UX, UI, interactive design techniques and software to communicate a concept to an audience. Students will understand advanced interactive design techniques and apply techniques to class assignments. Students will create content-rich design solutions showcased within an interactive context using working prototypes in mobile-first design. Prerequisites:DDSN 1100,1101,1102,1401,2100,2101,2600,2600,2601,2602,CPST 3400,CPST 3410.Prerequisite(s):DDSN 1400.
